Smali â€“ Android's Assembly Language

Smali is to Android what assembly is to native binaries.

Smali Code
if-eqz v0, :cond_0
invoke-virtual {p0}, Lcom/app/Main;->unlock()V
Translation

If v0 == 0, jump

Otherwise, unlock feature

Readable. Logical. Patchable.

Smali Patching Example

Original Logic
if-eqz v0, :fail
; If v0 == 0, jump to fail
Patched Version
const/4 v0, 0x1
; Set v0 = 1 (always true)

This teaches:

  • Control flow manipulation
  • Boolean logic understanding
  • App trust assumptions

Rebuilding and Signing APK

1. Decompile with apktool
apktool d app.apk
2. Modify Smali code

Edit the .smali files in the decompiled directory

3. Rebuild APK
apktool b app
4. Sign APK
jarsigner -keystore debug.keystore app.apk androiddebugkey
5. Install
adb install app.apk

Key Mental Models

Model #1: Apps Trust Too Much

Many apps assume:

  • Users won't look inside
  • Logic won't be modified
  • Checks are safe locally

Reverse engineering teaches why this assumption fails.

Model #2: UI Is Not Logic

Buttons and screens mean nothing. Logic lives in:

  • Methods
  • Conditions
  • Native calls

Always follow code paths, not UI paths.
